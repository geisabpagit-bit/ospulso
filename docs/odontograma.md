# ✅ Checklist técnico – Odontograma digital México

## 1. Normatividad y legalidad
- [ ] Cumple con **NOM-013-SSA2-2015** (prevención y control de enfermedades bucales).
- [ ] Cumple con **NOM-004-SSA3-2012** (expediente clínico electrónico).
- [ ] Conservación mínima de registros: **5 años**.
- [ ] Incluye **firma digital** del odontólogo responsable.

## 2. Base de datos y unicidad
- [x] Cada odontograma está ligado a un **ID único de paciente**.
- [x] Se utiliza **numeración FDI** (dos dígitos por pieza dental).
- [x] Registro de **superficies dentales**: oclusal, mesial, distal, vestibular, lingual.
- [ ] Historial de tratamientos con **fecha, procedimiento y odontólogo**.
- [ ] Compatible con **DICOM/HL7** para interoperabilidad hospitalaria.

## 3. Interfaz gráfica interactiva
- [x] Mapa dental completo: **dentición permanente y decidua**.
- [x] Botones funcionales: **caries, corona, extracción, sano, implante, prótesis**.
- [x] Selector de vista: **adulto/niño**.
- [ ] Exportación en **PDF** vinculada al expediente clínico.
- [x] Registro dinámico con **trazabilidad de cambios**.

## 4. Seguridad y accesibilidad
- [ ] Roles de usuario diferenciados: **odontólogo, asistente, administrador**.
- [ ] Autenticación segura: **usuario/contraseña** (biometría opcional).
- [ ] Cifrado de datos: **AES-256**.
- [ ] Respaldo en la nube con **redundancia**.
- [ ] Cumple con la **Ley Federal de Protección de Datos Personales (LFPDPPP)**.

## 5. Funciones adicionales
- [ ] Alertas clínicas: **tratamientos pendientes y revisiones programadas**.
- [ ] Integración con **radiografía digital** (RVG, OPG).
- [ ] Compatibilidad **multi-sucursal** con acceso remoto seguro.
- [ ] Reportes estadísticos: **prevalencia de caries, tratamientos realizados**.
